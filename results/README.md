## How to Navigate

Each test run folder contains:
- `index.html` - Interactive dashboard with charts and navigation
- `markdown/` - Detailed text reports for each persona/language
- `assets/` - Stylesheets and scripts for dashboard
- `README.md` - Overview and methodology for that specific test

## Folder Naming Convention

Folders are named by timestamp: `YYYY-MM-DD_HH-MM-SS`

This allows you to:
- Easily track when tests were run
- Compare results across different test runs
- Maintain historical record of research
