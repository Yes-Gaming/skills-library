# CTA Testing Skill

**Systematic methodology for testing call-to-action (CTA) copy and button text with synthetic gambling personas.**

## Overview

This skill provides a structured framework for testing CTA effectiveness across different contexts, personas, and use cases. Unlike tagline testing, CTA testing is highly context-dependent and focuses on conversion optimization rather than brand preference.

## Key Differences from Tagline Testing

| Aspect | Tagline Testing | CTA Testing |
|--------|----------------|-------------|
| **Focus** | Brand preference, appeal | Conversion, action intent |
| **Context** | Optional, general | Critical, must be specific |
| **Key Metric** | Preference votes | Intent-to-value ratio |
| **Friction** | Less critical | Make-or-break factor |
| **Versatility** | Taglines work broadly | CTAs are context-specific |
| **Testing** | Can test standalone | Must test in context |

Key metrics to examine:
- **Action Intent (Q1):** Who would click?
- **Motivation (Q2):** Why would they click?
- **Friction (Q3):** What stops them?

## Testing Contexts

### 1. Landing Page Hero
- **User State:** New visitor, evaluating options
- **Goal:** Initial sign-up conversion
- **Test:** "Start Playing Free" vs "Join Now" vs "Get Started"

### 2. Registration Flow
- **User State:** Mid-registration, deciding to complete
- **Goal:** Reduce drop-off
- **Test:** "Create Account" vs "Continue" vs "Almost Done"

### 3. First Deposit
- **User State:** Registered, evaluating trust
- **Goal:** First deposit conversion
- **Test:** "Make First Deposit" vs "Add Funds" vs "Deposit & Play"

### 4. Promotional Email
- **User State:** Existing user, evaluating offer
- **Goal:** Engagement, promotional uptake
- **Test:** "Claim Bonus" vs "Get Offer" vs "Activate Now"

### 5. Re-engagement
- **User State:** Dormant user, considering return
- **Goal:** Reactivation
- **Test:** "Welcome Back" vs "Return & Play" vs "Come Back"

### 6. Mobile In-App
- **User State:** Active session, mid-activity
- **Goal:** Continued engagement
- **Test:** "Play Now" vs "Bet Now" vs "Go"
